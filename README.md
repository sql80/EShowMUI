EShowMUI 为移动 Web 服务量身设计
====



## 概述

EShowMUI 是一套适用于微信端的html静态资源，由ForDesign.Me设计团队为到群网 移动Web 开发量身设计，可以令用户的使用感知更加统一。
包含`button`、`cell`、`dialog`、 `toast`、`article`、`icon`等各式元素。
